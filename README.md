# AOV

Dự án AOV — mô tả ngắn gọn về mục tiêu và phạm vi của dự án.

## Giới thiệu

AOV là dự án [mô tả mục đích dự án tại đây]. Dự án được tổ chức theo cấu trúc rõ ràng để dễ dàng quản lý tài liệu, dữ liệu và báo cáo.

## Cấu trúc thư mục

```
AOV/
├── README.md        # Giới thiệu dự án (file này)
├── CLAUDE.md        # Hướng dẫn cho Claude Code
├── docs/            # Tài liệu dự án
├── data/            # Dữ liệu thô và đã xử lý
└── reports/         # Báo cáo và kết quả
```

## Bắt đầu

1. Đọc tài liệu trong thư mục `docs/`
2. Dữ liệu đầu vào đặt trong `data/`
3. Kết quả và báo cáo lưu trong `reports/`
